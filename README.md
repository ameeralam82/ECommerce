# ECommerce
A full-stack web-based e-commerce application that allows users to browse products, add items to a cart, and place orders. The system also includes an admin panel for managing products and categories.

## Features
- Product search and filtering
- View product specifications, images, and reviews
- Add to cart, edit cart, and checkout
- Admin panel for product and category management
- Secure database-driven architecture

## Tech Stack
- PHP
- MySQL
- HTML, CSS
- JavaScript

- Cart
- Orders
- Reviews

## How to Run Locally
1. Install XAMPP / WAMP
2. Clone the repository
3. Import the database `.sql` file into phpMyAdmin
4. Place the project in the `htdocs` folder
5. Start Apache & MySQL
6. Open `http://localhost/project-folder`
